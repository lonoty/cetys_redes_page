PRTG

Introducción 

Subtitle

PRTG es un programe que te permite monitorear dispositivos y servicios en la red para ver el estado de una compnaia o una red en si. 
Lo que se busco monitorear por medio de [rtg fue lo siguiente:
-Uptime
-Uso de CPU
-Memoria disponible
-Espacio en disco disponible
-Trafico de cada adaptador de red (Ancho de banda).

También se lleva a cabo un monitoreo de los siguientes servicios (si es aplicable para dicho host): 
-HTTP
-SSH
-HTTPS
-DNS
-freeRadius
-SMTP
-POP3
-IMAP

Configuración

Para poder monitorear estos servicios se tiene que registra los dispositivos a medir para ello se tiene que identificar los IP de cada equipo, y registrarlo como new device. 

Dependiendo del equipo y de que servicios tiene se tendrá que agregar los sensores correctos, para esto se tiene que saber si la maquina que se va a monitorear tiene SNMP o que puertos puede revisar. Una vez teniendo esta información se crea un sensor como se demuestra en la imagen.

Cada sensor tiene sus configuraciones únicas ya que depende de que servicio se esta monitoreando. Para los servicios que se van a monitorear por SNMP se tienen que configurar para que el community-string sea el mismo que el del agente.
